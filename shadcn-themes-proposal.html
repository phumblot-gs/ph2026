  • • Composants copiés dans votre code (pas de dépendance)

  • • Personnalisation totale avec Tailwind
  • • TypeScript natif
  • • Mode sombre intégré
  • • Performance optimale
  • • Documentation excellente

  • • Charts (avec Recharts)
  • • Dialogs, Sheets, Toasts
  • • Navigation (Tabs, Breadcrumb)
  • • Data Display (Avatar, Badge, Card)
  • • Et 40+ autres composants
